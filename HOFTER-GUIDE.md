## 1. 什么是 Hofter

Hofter 是一个运行在 Roche 平台上的同人小说社区插件。你可以在这里：

- **发现** 基于你喜欢的 CP 和标签自动生成的同人文
- **阅读** 沉浸式的多章节同人文，支持段评、追更
- **创作** 灵感创作或手写创作，发布你的同人文
- **互动** 与角色评论互动，发表你的看法
- **润色** 使用 AI 校正润色你的文笔

所有数据都存储在你的设备本地，不会上传到任何第三方服务器。

## 2. 首次使用：开屏设置

首次打开 Hofter 时，你需要完成以下设置：

### 2.1 选择人设

从你在 Roche 上创建的人设中选择一个作为你在 Hofter 中的身份。这个人设将影响同人文的生成风格和评论互动。

> 如果没有看到你的人设，请先在 Roche 中创建一个人设，然后返回 Hofter 重新打开人设切换面板。

### 2.2 选择角色

选择你想要组 CP 的角色。你可以选择 1-2 个角色来组成 CP。

### 2.3 添加标签

为你的 CP 添加标签（如"先婚后爱"、"破镜重圆"等），这些标签会影响同人文的情节走向。

### 2.4 完成设置

点击"开始探索"进入首页。之后你可以随时在设置中修改这些选项。

---

## 3. 首页：发现与探索

### 3.1 信息流

首页会根据你的 CP 和标签自动生成同人文摘要卡片。每张卡片显示：

- **标题** 和 **CP 名**
- **内容摘要** 和 **标签**
- **作者信息**（如果是你创作的会标注"你"）

### 3.2 刷新内容

下拉首页可以刷新，获取新的同人文摘要。

### 3.3 筛选与搜索

- 点击顶部的 **CP 标签** 可以切换不同 CP 的内容
- 点击 **标签探索** 可以发现更多标签

### 3.4 摘要卡片操作

- **点击卡片**：进入阅读页面
- **长按卡片**：进入复选模式，可以批量选择生成正文
- **右下角按钮**：收藏、稍后读

---

## 4. 阅读同人文

### 4.1 生成正文

点击摘要卡片后，如果还没有生成正文，Hofter 会自动为你生成完整的同人文。生成过程中会显示加载动画。

### 4.2 翻页阅读

正文按章节分页显示。左右滑动或点击底部的翻页按钮切换章节。

### 4.3 段评

正文中某些段落旁边会出现小标记，表示该段落有 **段评**（类似旁注）。点击标记可以查看其他读者/角色对该段落的评论。

### 4.4 阅读设置

点击右上角的设置图标可以调整：

- **字体大小**
- **主题**（浅色/深色）

### 4.5 更多操作

点击右上角"更多"按钮，可以：

- **重新生成正文**：对当前章节重新生成（翻到哪章就重roll哪章）
- **追更续章**：继续生成下一章
- **校正润色**：对正文进行文笔润色
- **查看原文**：查看润色前的原始正文
- **查看模型上下文**：查看发送给 AI 的完整提示词（调试用）

## 5. 评论区

### 5.1 评论类型

评论区包含两种评论：

- **普通评论**：独立发表的评论
- **段评**：针对正文中特定段落的旁注评论

### 5.2 角色评论

当你在阅读时，你 CP 中的角色可能会自动出现在评论区发表评论！角色评论带有 **角色** 标记和角色头像。

角色评论的特点：
- 评论数量由角色性格决定（1-5条不等）
- 有时会回复你的评论，与你互动
- 评论内容会写入角色的记忆，影响后续互动

### 5.3 发表评论

在评论区底部的输入框中输入你的评论，点击"发送"即可。发送评论后，系统会自动生成新的评论与你的评论互动。

### 5.4 回复评论

点击任意评论可以回复该评论。回复会显示"回复 XXX"的标记。

### 5.5 刷新评论

点击评论区右上角的刷新按钮可以重新生成评论。新评论会出现在评论列表顶部。

### 5.6 评论折叠

- **回复折叠**：一条评论下的多条回复默认折叠，点击"展开 X 条回复"查看
- **评论区折叠**：超过 3 条评论时，只显示前 3 条，点击"查看更多评论"展开全部

### 5.7 删除评论

点击评论右下角的"删除"按钮可以删除评论。需要点击两次确认（防误触）。

### 5.8 翻译评论

角色评论旁边可能显示"译"按钮，点击可以查看评论的中文/英文翻译。

## 6. 追更续写

### 6.1 续写新章节

阅读完已有章节后，点击"追更续章"可以生成下一章。你可以选择：

- **发送给模型的前文信息**：完整正文 或 内容总结
  - 完整正文：模型能看到全部前文，续写更连贯，但消耗更多 token
  - 内容总结：模型只看摘要，续写可能偏离细节，但更省 token
- **给作者提意见**：可选，写下你对前文的看法或建议（如"希望更甜一点"）

### 6.2 多章节管理

- 续写后章节会自动追加
- 翻页可以查看不同章节
- 重新生成时，翻到哪章就重roll哪章及之后的章节

## 7. 校正润色

### 7.1 什么是校正润色

校正润色是 Hofter 的 AI 文笔优化功能。它会在不改变故事情节的前提下，提升文本的文学质量，包括：

- 拆分文字墙，优化段落节奏
- 去除程度副词滥用（"极其""无比"等）
- 将旁白直接命名的角色特征改为行为呈现
- 消除元旁白翻译句式
- 降格剧烈反应描写
- 禁止身体零件化描写
- 优化环境描写与角色的连接

### 7.2 自动润色

在 **设置 → 校正润色** 中开启"自动润色"后，每次生成正文时会自动进行润色。

> 注意：自动润色只在单章时生效。如果你已经追更了多章，自动润色会自动跳过（避免影响后续章节一致性）。

### 7.3 手动润色

在阅读页面点击 **更多 → 校正润色** 可以手动触发润色。

> 同样，手动润色也只支持单章正文。多章追更后不支持润色。

### 7.4 润色模式

在设置中可以选择两种润色模式：

- **直接返回**：模型直接返回修改后的完整正文，速度快
- **内联校验**：模型逐段思考、校验、修改，质量更高但速度较慢

### 7.5 润色预设

你可以创建多个润色预设，每个预设包含不同的润色规则：

1. 进入 **设置 → 校正润色 → 管理润色预设**
2. 每个预设由三部分组成：
   - **身份定义**（不可编辑）：定义 AI 的角色和任务
   - **润色与校正规则**（可自由编辑）：具体的润色要求
   - **返回格式**（不可编辑）：规定 AI 的输出格式
3. 你可以添加、编辑、删除、切换预设
4. 点击"添加润色预设"创建新预设，修改中间的润色规则部分即可

### 7.6 保留原始正文

在设置中开启"保留原始正文"后，每次润色前的原始正文会被保存（最多保留最近 5 次）。你可以在阅读页面点击 **更多 → 查看原文** 查看历史版本。

### 7.7 润色专用模型

你可以为润色指定一个不同于正文生成的模型预设：

1. 进入 **设置 → 校正润色 → 管理润色预设**
2. 在底部的"润色专用模型"下拉框中选择一个已配置的轻型模型预设
3. 留空则使用当前默认的轻型模型

## 8. 创作同人文

### 8.1 灵感创作

灵感创作是 AI 辅助的创作方式：

1. 点击首页右下角的 **+** 按钮
2. 选择 **灵感创作**
3. 选择 CP 和标签
4. 可选：输入你的灵感/想法
5. 点击"开始创作"，AI 会为你生成完整的同人文
6. 生成后可以编辑标题，然后发布

### 8.2 手写创作

手写创作是你自己撰写正文的方式：

1. 点击首页右下角的 **+** 按钮
2. 选择 **手写创作**
3. 选择 CP 和标签
4. 输入标题
5. 在正文编辑区直接撰写你的同人文
6. 完成后点击"发布"

### 8.3 动态发布

动态发布是简短的文字分享：

1. 点击首页右下角的 **+** 按钮
2. 选择 **动态**
3. 输入你想分享的内容
4. 点击"发布"

### 8.4 创作后自动操作

在设置中开启以下选项后，创作完成时会自动执行：

- **自动润色**：生成正文后自动润色（需开启自动润色）
- **自动生成评论**：生成正文后自动生成读者评论和段评

## 9. 我的页

### 9.1 个人信息

显示你的人设头像和名称，以及发布数、收藏数、稍后读数。

### 9.2 标签页

点击头像或名称可以打开人设切换面板，切换到不同的 Roche 人设。

## 10. 设置详解

### 10.1 生成设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 自动生成评论 | 生成正文后自动生成评论和段评 | 关闭 |
| 自动关注标签 | 自动关注你探索过的标签 | 关闭 |
| 预设语言 | 正文和评论提示词的语言（中文/English） | 中文 |
| 字数范围 | 生成正文的目标字数范围 | 3000-8000 |

### 10.2 校正润色

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 自动润色 | 生成正文后自动润色 | 关闭 |
| 润色模式 | 内联校验（逐段）/ 直接返回 | 直接返回 |
| 保留原始正文 | 润色前保留原始正文（最多5次） | 关闭 |
| 润色预设 | 管理润色规则预设 | 默认预设 |
| 润色专用模型 | 为润色指定不同的模型预设 | 使用默认模型 |

### 10.3 轻型模型

配置用于同人文生成、评论生成、润色等功能的 AI 模型。

- **添加预设**：输入 API 地址、API Key、模型名称
- **测试连接**：保存前可以测试 API 是否可用
- **获取模型列表**：自动获取该 API 支持的模型列表
- **切换预设**：点击预设右侧的"启用"按钮

> 支持所有兼容 OpenAI API 格式的模型服务（如 OpenAI、DeepSeek、硅基流动等）。

### 10.4 显示设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 主题 | 浅色/深色 | 浅色 |
| 字体大小 | 阅读页面的字体大小 | 17 |

### 10.5 CP 管理

- 查看已添加的 CP 标签
- 删除不需要的 CP 标签
- 添加新的 CP 标签

### 10.6 记忆设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 记忆附加概率 | 角色评论时附加记忆的概率 | 30% |
| 分享记忆模式 | 分享同人文时是否附带角色记忆 | 自动 |

### 10.7 关于

显示 Hofter 版本信息和开发信息。

## 11. 分享功能

### 11.1 分享悬浮球

Hofter 提供一个全局的分享悬浮球，你可以在任何页面拖拽使用。

### 11.2 分享同人文

1. 点击分享悬浮球
2. 选择要分享的同人文
3. 选择分享方式：
   - **分享到聊天**：将同人文分享到 Roche 的聊天中
   - **复制链接**：复制分享链接

### 11.3 分享卡片

分享到聊天时会生成精美的分享卡片，包含：

- 同人文标题和 CP 名
- 内容摘要
- 标签
- 作者信息

## 12. 常见问题

### Q: 为什么看不到我刚在 Roche 创建的人设？

打开人设切换面板时，Hofter 会重新获取你的人设列表。如果仍然看不到，请确认人设已在 Roche 中保存成功。

### Q: 人设切换面板无法滚动？

面板支持滚动，如果内容较多请向上滑动查看所有人设。

### Q: 角色评论为什么没有出现？

角色评论需要满足以下条件：
- 当前会话的角色在短期记忆中看到了 [HF] 标记
- 评论生成是异步的，可能需要等待几秒
- 系统会检查最近 150 条短期记忆，避免角色过度关注

### Q: 评论刷新后没有与之前的评论互动？

评论互动依赖于已有评论的传递。手动刷新评论时会将已有评论传给 AI，确保新评论能与旧评论互动。如果你发了评论后触发的自动刷新，系统会特别强调至少 2 条评论与你的评论互动。

### Q: 润色后段评位置不对了怎么办？

润色后系统会自动尝试用段评的引用文本重新匹配位置。如果匹配不上，该段评会自动隐藏，不会显示在错误的位置。

### Q: 为什么多章追更后不能润色了？

润色会修改正文内容，如果已经追更了多章，润色第一章可能导致后续章节的衔接不一致。所以多章情况下润色功能会自动禁用。

### Q: 批量生成正文怎么用？

1. 在首页长按一张摘要卡片进入复选模式
2. 勾选你想要生成正文的摘要卡片
3. 点击"一键生成"
4. 系统会退出复选模式，在后台逐个生成正文
5. 顶部会显示进度条，你可以随时点击"停止"取消
6. 生成过程中你可以自由浏览其他内容

### Q: 重新生成正文是重roll全部还是当前章节？

取决于你当前翻到哪一章：
- 翻到第 1 章 → 重新生成全部正文
- 翻到第 3 章 → 只重新生成第 3 章及之后的章节，第 1-2 章保持不变

### Q: 如何自定义润色规则？

1. 进入 设置 → 校正润色 → 管理润色预设
2. 点击"添加润色预设"或编辑已有预设
3. 修改中间的"润色与校正规则"部分
4. 你可以参考默认模板，按需调整规则
5. 保存后切换到该预设即可生效

### Q: 数据安全吗？

Hofter 的所有数据都存储在你的设备本地（Roche 的本地存储），不会上传到任何第三方服务器。你配置的 API Key 仅用于向你指定的 AI 服务发起请求，不会经过任何中间服务器。
